# 📋 React Todo App

A simple and interactive Todo List application built with React. This app allows users to:
- Add new tasks
- Mark tasks as completed
- Delete tasks

## 🚀 Features
- ✅ Add new todos
- ✏️ Mark todos as complete/incomplete
- 🗑️ Delete todos
- 🎨 Clean, responsive UI

## 🛠️ Tech Stack
- **React**
- **CSS**
- **UUID** (for unique ID generation)
